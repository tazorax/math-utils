This library provide mathematical objects for PHP 5.4+

## Content
### 2D (WIP)
- Point2d
- Polygon
- Rectangle
- Triangle
- Utils
- Vector2d

### 3D (WIP)
- Point3d
- Vector3d
- Matrix3d

### Arithmetic
- Fraction
- FractionCollection
- Utils

## Install
Using composer

``` composer require tazorax/math-utils ```
