Mathematical objects for PHP 5.4, 5.5, 5.6, 7.0 and HHVM

## Content
### 2D (WIP)
- Point2d
- Triangle
- Vector2d

### 3D (WIP)
- Point3d
- Vector3d
- Matrix3d

### Arithmetic
- Fraction
- FractionCollection
- Utils

## Install
Using composer

``` composer require tazorax/math-utils ```
